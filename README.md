# APPTOPIA Landing Page

This is the marketing landing page for **APPTOPIA**, built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting started

You’ll need Node.js and npm installed.

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

Then open the URL shown in your terminal (typically `http://localhost:5173/`) in your browser.

## Customization

- Update the document title and meta tags in `index.html`.
- Edit the React pages and components under `src/` to change layout, copy, and styling.

You’re free to adapt this project’s content, branding, and structure to match your own product or company.
